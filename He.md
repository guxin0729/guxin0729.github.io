#### pwd                           显示当前目录的文件和文件夹
#### git status                    显示当前状态
#### touch 文件名.格式             创建一个文件
#### git commit -m"名字"           设置节点(上传前需设置)
#### git push -u origin master     传送到git
#### ssh-keygen -t rsa -C "邮箱"   如果之前有其他用户连接过git ,需要这步重新设置
#### git clone 文件链接(选择ssh)   拷贝到本地
#### 密钥在 C:\Users\Administrator\.ssh\id_rsa.pub
#### git add 文件名.格式           添加到本地库
#### mkdir 名字                    创建文件夹
